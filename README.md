## Development
### Requirements
- [bun (v1.2.22 and up)](https://bun.com/)
- Basic React/JS knowledge

### Preparing
```sh
# Clone project
$ git clone https://github.com/meowabyte/bsky-bookmarks-import
# Install dependencies
$ bun install
# Run build script (use --watch for dev mode)
$ bun run build
```
